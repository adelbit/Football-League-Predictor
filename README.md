We used datasets from kaggle and the following tech stack:-
Tech used: 
1) Jupyter Notebook
2) Python 3.8
3) Pandas 
4) NumPy
5) Scikit-Learn 
6) TensorFlow 2.0
7) Keras 
8) Seaborn 
9) MatplotLib
